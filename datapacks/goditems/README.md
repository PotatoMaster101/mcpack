# God Items
Gives god items (ruins the fun).

## Usage
To get armor:
```
/function goditems:armor
```
To get weapons:
```
/function goditems:weapons
```
To get potions:
```
/function goditems:potions
```
To get tools:
```
/function goditems:tools
```
To get player killing potions:
```
/function goditems:death
```
