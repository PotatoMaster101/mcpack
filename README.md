# Minecraft Data Packs
Some data packs I made to make Minecraft more interesting.

## Install (Manual)
1. Go to the world save folder that you want to install data packs
2. Drag and drop the `datapacks` folder into the world save folder (replace existing)
3. If you are already playing the on the world save, use the `/reload` command

## Install (Script)
1. Run `Install-DataPack.ps1` and specify the world save name (requires at least [PowerShell Core](https://github.com/PowerShell/PowerShell) 7 to run)
2. If you are already playing the on the world save, use the `/reload` command
